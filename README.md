# NumPy Practice / Projects

A collection of NumPy exercises, examples, and small projects to master numerical computing in Python.

## 📌 About

This repository contains various Python scripts and Jupyter notebooks demonstrating different concepts and functionalities of **NumPy** - the fundamental package for scientific computing in Python.

I created this while learning/practicing NumPy. It includes basic operations, advanced techniques, performance comparisons, and real-world use cases.

## ✨ Features

- Basic array operations
- Mathematical and statistical functions
- Broadcasting and vectorization
- Array manipulation and reshaping
- Linear algebra operations
- Random number generation
- Performance optimization examples
- Real-world data processing examples

## 🛠️ Technologies Used

- **Python 3.8+**
- **NumPy**
- Jupyter Notebook (optional)
